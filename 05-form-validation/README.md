# 05 - Form Validation

A "Create Account" form with client-side validation and inline error UI.

## Validations
- Name: required, min 2 characters
- Email: required, must be valid format
- Password: required, 8+ chars, includes 1 letter + 1 number
- Confirm password: must match
- Terms: must be checked

## UX
- Inline errors + invalid styling
- Live validation while typing
- Simulated submit (loading state) + success banner

Tech: HTML, CSS, JavaScript
