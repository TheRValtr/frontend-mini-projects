# 06 - Weather App

A vanilla JS weather app demonstrating async UI patterns:
- Fetch API
- Loading state
- Error state + retry
- Basic geocoding (city -> lat/lon)
- Unit toggle (°F/°C)

APIs (no key required):
- Open-Meteo Geocoding
- Open-Meteo Forecast

Tech: HTML, CSS, JavaScript

## How to run
Open `index.html` in a browser (internet required).
